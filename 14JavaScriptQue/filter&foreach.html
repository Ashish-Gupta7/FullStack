ForEach

forEach ek JavaScript method hai jo array ke har element par iterate karne ke liye use kiya jata hai. Yeh method ek callback function leta hai, jo har element par apply hota hai. Yeh method array ke original array ko modify nahi karta, toh original array ko change nahi karta.

Syntax : array.forEach(function(currentValue, index, arr), thisValue)

currentValue (required): Yeh parameter woh element hai jo currently iterate hua hai.

index (optional): Yeh parameter woh index hai jo currently iterate hua element ke liye hai.

arr (optional): Yeh parameter woh array hai jo currently iterate hua hai.

thisValue (optional): Yeh parameter this keyword ke liye use kiya jata hai callback function mein.

Filter

filter ek JavaScript method hai jo ek array se selected elements ko ek nayi array mein return karta hai. Yeh method ek callback function leta hai, jo har element par apply hota hai. Yeh method sirf woh elements ko select karta hai jo callback function ke through true return karta hai.

Syntax : array.filter(function(element, index, array), thisValue)

element (required): Yeh parameter woh element hai jo currently iterate hua hai.

index (optional): Yeh parameter woh index hai jo currently iterate hua element ke liye hai.

array (optional): Yeh parameter woh array hai jo currently iterate hua hai.

thisValue (optional): Yeh parameter this keyword ke liye use kiya jata hai callback function mein.

Includes

includes ek JavaScript method hai jo ek array ya string mein kisi specific element ya value ke presence ko check karta hai. Yeh method true ya false return karta hai, depend karta hai ki specified element ya value array ya string mein hai ya nahi.

Syntax : array.includes(element, start)

element (required): Yeh parameter woh element hai jo ki check kiya jata hai ki wo array ya string mein hai ya nahi.

start (optional): Yeh parameter woh index hai jisme search shuru hoga. Default value 0 hai.

Some

some ek JavaScript method hai jo ek array ke elements par iterate karke check karta hai ki kya kisi bhi element ne specified condition ko satisfy kiya hai ya nahi. Yeh method ek callback function leta hai, jo har element par apply hota hai. Agar kisi bhi element ne condition ko satisfy kiya toh some method true return karta hai, nahi toh false return karta hai.

Syntax : array.some(function(element, index, array), thisValue)

element (required): Yeh parameter woh element hai jo currently iterate hua hai.

index (optional): Yeh parameter woh index hai jo currently iterate hua element ke liye hai.

array (optional): Yeh parameter woh array hai jo currently iterate hua hai.

thisValue (optional): Yeh parameter this keyword ke liye use kiya jata hai callback function mein.

some ko jaha pass bhi true milta hai ye wahi se return ho jata h.

Example

function checkArrayForObject(arr, obj) {

// Check if any element of the array matches the object

return arr.some(item => {

console.log(item);

// Check if all key-value pairs of the object match the current element

console.log(Object.keys(obj));

return Object.keys(obj).every(key => {

return item.hasOwnProperty(key) && item[key] === obj[key];

});

});

}

// Example usage:

const myObject = {c: 3, d: 4};

(checkArrayForObject(myArray, myObject)); // Output: true

yaha myArray aur myObject ki tulna krne pr 1st index pr value match ho jati h iss wajah se uper console.log(item); pr myArray ka last index {e: 5, f: 6} console pr print nhi ho rha h, yadi aap myObject ki value ko {c: 3, d: 4}; isse badalkar {e: 5, f: 6} ise krte ho tb some sabhi ko check karega kyuki ye myArray ki last value h.

Difference between some() and filter()

some() method sirf check karta hai ki kya array mein kisi bhi element par condition true hoga, aur iske baad true ya false return karta hai. // filter() method ek naya array return karta hai jismein sirf woh elements hain jo condition par pass hote hain.

Every

every method ek array ke saare elements ko check karne ke liye use hota hai. Yeh method ek callback function leta hai, jo har element par apply hota hai. Agar callback function har element ke liye true return karta hai, toh every method true return karega. Yani, every method check karta hai ki kya array ke saare elements callback function ke conditions ko satisfy kar rahe hain ya nahi.

Syntax : array.every(function(currentValue, index, arr), thisValue)

array: Yeh array hai jiska elements check kiya jayega.

function(currentValue, index, arr): Yeh ek callback function hai jo har element par apply hota hai.

currentValue: Yeh parameter current element ko represent karta hai.

index (optional): Yeh parameter current element ke index ko represent karta hai.

arr (optional): Yeh parameter array ko represent karta hai.

thisValue (optional): Yeh parameter callback function ke this keyword ke liye use hota hai.

Reduce

reduce method ek array ke saath kaam karne ke liye use hota hai. Yeh method ek function leta hai jisme hum ek accumulator (acc) aur current element (elm) specify kar sakte hain. Yeh function array ke saath iterate karne ke baad, accumulator mein current element ke value ko add kar deta hai.

Syntax :

array.reduce(function(acc, elm, index, array) {

// yeh function har ek element par iterate karne ke baad call hota hai

// acc - accumulator, ek variable hai jisme hum result store kar sakte hain

// elm - current element, array ke saath iterate karne wale element

// index - current element ke index

// array - original array

return acc + elm; // yeh example hai, jisme hum accumulator mein current element ko add kar rahe hain

}, initialValue); // initialValue - yeh optional hai, jisme hum accumulator ke initial value specify kar sakte hain
