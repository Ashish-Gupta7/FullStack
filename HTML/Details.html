Html(Hyper text markup language) ke bina koi bhi web site exist hi nhi karti h, html koi programming language nhi h ye ek markup language h, markup language ek aisi language h jo tags ki madad se browser ko instructions dete h aur uss hisab se ye language tags ko web page pr display karta h. html bahot saare elements se bani h. tag se milkar element banta h.

Tag :-

tag do tarah ke hote h, inhe less than aur greater than ke beech me likha jata h, inhe browser samajhta h.

1. opening tag -

it does not has any range. inhe less than aur greater than ke beech me likha jata h.
Ex. br tag, hr tag, img tag...

2. closing tag -

it has a range. inhe less than forward slash aur greater than ke beech me likha jata h
Ex. html tag, p tag, b tag, div tag, body tag, head tag...

Nesting of elements :

Ek element ke andar kai saare elements uske andar bhi elements ise hi nesting of element kahte h.
yadi hm sirf elements ki madad se web site banaye to wo ek basic text wali web lagegi isliye ise intresting banane ke liye attributes ka use kiya jata h.

Attributes :

har elements ko aditional properties dene ke liye attributes ka use kiya jata h.
Ex. 1. image ki width and height manually set kr set h.
2. color set kr skte h.
